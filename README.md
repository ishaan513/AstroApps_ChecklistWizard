# AstroApps_ChecklistWizard
Part of the Astro App suite. Checklist Wizard allows users to input procedural checklists, manage them, and use them during operations.

# User's Guide
1. Enter your name in the side menu to log all checks
2. If creating a new checklist template, set mode to *Manage Templates*
3. If you are working in an active checklist, set mode to *View Active Checklists*
4. If you are creating a new instance of a checklist template, set mode to *Start New Checklist*
5. Within *Manage Checklists*, you can create a new checklist template. New items should be separated with new lines. Required items should start with *
6. Within *View Active Checklists*, you can select an active checklist, check off items, and close out the checklist.
7. Within *Start New Checklist*, you can create a new instance of a checklist by first selecting the template and then entering a new name. Once you hit *Start Checklist*, the page will not change, but the new list will be available in *View Active Checklists*
